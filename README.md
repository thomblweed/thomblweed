# thomblweed

## Prerequisites

### Install Bun

https://bun.sh/docs/installation

### Install Docker Desktop

https://www.docker.com/products/docker-desktop/

### Install Taskfile

```sh
bun add --global @go-task/cli
```

## Development

### Install dependencies

```sh
bun install
```

### development

```sh
task dev
```
