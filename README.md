## Setup

### Install pnpm

https://pnpm.io/installation

### Install Docker Desktop

https://www.docker.com/products/docker-desktop/

### Install Taskfile

https://taskfile.dev/installation/

### Install Sops

https://github.com/getsops/sops

### Install Age

https://github.com/FiloSottile/age

```sh
task decrypt

pnpm install
```

task dev
